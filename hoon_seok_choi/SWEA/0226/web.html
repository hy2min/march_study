hyeprtext 특징 세가지 : 1. 비선형성 2. 상호연결성 3. 사용자 주도적 탐색 css 선택자는 전체, 요소, 클래스, 속성 으로 나뉜다. 전체 선택자는 *로 표기하며 모든 요소를 선택한다. 요소 선택자는 요소 이름으로 표기하며 해당 요소를 선택한다. 클래스 선택자는 .으로 표기하며 해당 클래스를 선택한다. 속성 선택자는 []로 표기하며 해당 속성을 선택한다. 자식 결합자는 특정 요소의 바로 한 단계 아래의 요소만 선택하는 결합자이고 ()로 표기한다. 자손 결합자는 특정 요소의 모든 하위 요소를 선택하는 결합자이고 > 로 표기한다. 명시도는 선택자의 중요도를 결정하기 위한 알고리즘 명시도의 순서는 1. !important 2. 인라인 스타일 3. 아이디 선택자 4. 클래스 선택자, 속성 선택자, 가상 클래스 선택자 5. 요소 선택자, 가상 요소 선택자 6. 전체 선택자 7. 상속된 스타일 명시도가 같을 경우 나중에 선언된 스타일이 우선순위를 가진다. css에서 상속되는 속성은 글꼴 관련 속성(font-family, font-size, font-weight, font-style, color) 텍스트 관련 속성(text-align, text-decoration, text-transform, line-height, letter-spacing, word-spacing) 상속되지 않는 속성은 박스 모델 관련 속성(display, position, float, clear, visibility, z-index) 포지션 관련 속성(top, right, bottom, left) 부트스트랩에서 reset css는 기본적인 스타일을 초기화하는 css이다. reset css를 사용하면 브라우저마다 다른 기본 스타일을 초기화하여 동일한 화면을 보여줄 수 있다. oocss는 객체지향 css로 스타일을 객체 단위로 분리하여 재사용성을 높이는 방법이다. 객체는 스타일과 마크업을 분리하여 스타일을 재사용할 수 있도록 한다.
